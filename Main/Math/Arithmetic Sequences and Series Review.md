# Arithmetic Sequences
- $d = \text{constant/common difference}$ $d = a_{n-1} - a$
- Explicit Formula: 
	- $a_{n} = a_{1} + d(n-1)$
	- $a_{n} = a_{0} + dn$
- Recursive Formula:
	- always have to have $a_{1}$ defined
	- $a_{n} = a_{n-1} + d$

# Arithmetic Series
- $$S_{n} = \frac{a_{1} + a_{n}}{2} \cdot n$$
- $$S_{n} = \frac{a_{1}+(a_{0}+d_n)}{2} \cdot n$$
